# 社交网络分析与智能推荐系统

## 1. 项目简介
本项目是针对《数据结构》课程设计的社交网络分析系统。系统通过自主实现的底层数据结构（哈希表、邻接表、堆），实现了社交关系的建模、人脉查询、最短社交距离计算以及基于兴趣标签的智能好友推荐。
开发语言：python 3.11
开发团队：李韦辰-2400130303;张昊楠-2400130306;杨宸轩-2400130304;
开发周期：2024-2025学年第二学期（1周/5个工作日）

## 2. 核心功能
* **人脉查询**：基于 **BFS（广度优先搜索）** 算法，实现一度（好友）与二度（可能认识的人）人脉查询。
* **社交距离**：计算任意两个用户之间的最短路径，展示社交链路。
* **智能推荐**：利用兴趣标签匹配度，结合 **最大堆** 结构进行 Top-K 好友推荐。

## 3. 技术栈
* **开发语言**：Python 3.11
* **UI 框架**：Tkinter 
* **数据结构**：
    * `HashTable`: 链地址法处理冲突，用于 $O(1)$ 级用户信息检索。
    * `AdjacencyList`: 邻接表实现无向图模型。
    * `MaxHeap`: 排序推荐结果。

##  5.功能说明
1.基础功能
•图结构建模：邻接表实现，支持用户、好友关系增删查改；
•数据持久化：加载CSV/TXT格式的用户及好友关系数据；
•一度人脉查询：输入用户ID，返回所有直接好友；
•二度人脉发现：基于BFS算法，查询好友的好友（排除自身及一度人脉）；
•社交距离计算：计算任意两用户最短社交距离及路径；
•用户信息管理：哈希表缓存用户信息，优化查询效率；
GUI界面：集成所有功能，支持输入校验、结果展示、异常提示。

2.扩展功能（加分项，已实现：智能推荐-基于共同兴趣的Top-3推荐，功能增强-实现多度人脉查询）

注意事项
•核心数据结构（邻接表、哈希表、堆）均为自主实现，禁止直接调用第三方库（如Python NetworkX、Java HashMap外的图库）；
•运行前请确保数据文件格式正确，避免解析异常；
•若扩展功能涉及多线程/数据库，需在环境配置中补充对应依赖；
•程序异常可查看控制台输出，常见问题：依赖未安装、数据路径错误、环境变量未配置。
